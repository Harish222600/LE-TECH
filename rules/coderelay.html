Code-Relay

Basic rules about this event:

Code Relay is a competitive debugging event where teams of four, each coding in different languages, race against the clock to fix code bugs in a relay-style challenge.
Brief Description:
Code Relay is an intense coding competition where multiple teams of four compete simultaneously. Each team member specializes in a different programming language and must debug their assigned code. The event follows a relay formatâ€”once a member completes their task, they pass the challenge to the next team member. The team that successfully fixes all bugs first wins. If no team completes all debugging tasks within 6 minutes, the team with the most fixed bugs within the time limit will be declared the winner and move to the next round.
Rules & Regulations:
-Team Composition: Each team consists of four members, each using a different programming language.
-Match Format: All teams compete at the same time.
-Debugging Task: Each team member must fix a bug in their respective coding language.
- Relay System: Once a team member completes their task, they pass the challenge to the next member and do not participate again in that round.
- Winning Criteria: The first team to successfully fix all bugs is declared the winner.
- Elimination: The teams with the least number of fixed bugs or the slowest completion time are eliminated.
- Fair Play: No external help (e.g., internet, AI tools) is allowed during the contest.
- Time Limit: If no team finishes within the time limit, the team with the most fixed bugs moves forward.
- Judging Panel: A panel of judges will verify the correctness of the fixes before confirming the winners.
